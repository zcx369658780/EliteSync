# 算法2.2：对拍与灰度运行手册（2026-03-29）

## 1) 对拍命令

后端新增命令：

```bash
php artisan app:dev:astro-shadow-compare --limit=50
```

可选参数：

- `--user-ids=1,2,3`：仅对指定用户对拍
- `--only-mismatch`：控制台只显示差异样本
- `--out=docs/devlogs/ASTRO_SHADOW_COMPARE.md`：Markdown 输出路径
- `--json=docs/devlogs/ASTRO_SHADOW_COMPARE.json`：JSON 输出路径

输出内容：

- 八字差异：`sun_sign / bazi / wu_xing`
- 西占差异：`sun/moon/asc/precision/engine/confidence_major`
- 能力缺口标记：`house_unsupported / aspect_unsupported`
- 每个用户的 old/new 结果与 diff 标记
- 摘要 Top 榜：
  - `top_diff_dimensions`（差异最多的维度）
  - `top_diff_users`（差异最多的样本用户）

## 2) 灰度开关配置

新增配置：`services/backend-laravel/config/astro_rollout.php`

### 八字灰度（bazi）

- 全局开关：`ASTRO_BAZI_CANONICAL_ENABLED`
- 白名单：`ASTRO_BAZI_CANONICAL_WHITELIST=9,12,33`
- 平台覆盖：
  - `ASTRO_BAZI_CANONICAL_ANDROID=legacy|canonical|''`
  - `ASTRO_BAZI_CANONICAL_IOS=...`
  - `ASTRO_BAZI_CANONICAL_WEB=...`
- profile_version 下限：`ASTRO_BAZI_MIN_PROFILE_VERSION`

### 西占灰度（western）

- 全局开关：`ASTRO_WESTERN_CANONICAL_ENABLED`
- 白名单：`ASTRO_WESTERN_CANONICAL_WHITELIST=...`
- 平台覆盖：
  - `ASTRO_WESTERN_CANONICAL_ANDROID=legacy|canonical|''`
  - `ASTRO_WESTERN_CANONICAL_IOS=...`
  - `ASTRO_WESTERN_CANONICAL_WEB=...`
- profile_version 下限：`ASTRO_WESTERN_MIN_PROFILE_VERSION`

---

## 3) 运行建议（上线前）

1. 全局保持 `western.enabled=false`，先小白名单跑对拍。
2. 每天生成对拍报告，观察 `any_diff_rate_pct` 走势。
3. 当 `any_diff_rate_pct`、`western_confidence_major_diff` 稳定后，再逐步放量。
4. `house_unsupported/aspect_unsupported` 目前是预期结果（高精引擎未接入）。

---

## 4) 说明

- 当前客户端不做 canonical 星盘计算，服务端为唯一真源。
- 低置信/降级模块在匹配解释中会自动降级措辞，避免“过强结论”。
