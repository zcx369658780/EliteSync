# EliteSync Changelog

当前活跃版本：`0.03.02a`

本文件是 EliteSync 的统一变更入口。历史单文件 changelog 已归档到 `docs/archive/legacy_2026-04/changelog/`。

## 0.03.02a - 2026-04-10

- 完成 3.2a 版本收口：星盘主视觉改为 APP 端本地完整绘制，盘面元素开关进一步细分到星体 / 虚点 / 相位 / 盘心，并支持恢复默认与预设档位切换。
- 完成 Android 宿主 bootstrap 修正，确保 API / WS 基线在启动 Flutter 前已注入，不再依赖 `slowdate.top` 的隐式回退地址。
- 版本号、版本检查、关于页历史、发布配置与 APK 文件名同步到 `0.03.02a / 30201`，用于继续归档和后续发布审计。

## 0.03.02 - 2026-04-10

- 完成 3.2 终验收尾与版本收口：补齐终验补证、归档材料与截图，核心链路正式归档。
- 星盘主视觉改为 APP 端本地完整绘制，并优化默认缩放与展示比例，避免圆盘被裁切。
- Android 宿主 bootstrap 已补齐 API / WS 基线注入，登录后不再依赖隐式回退地址，降低 `Request Timed out` 误报风险。

- 更新版本号、版本检查、关于页历史、发布配置与 APK 文件名到 `0.03.01 / 301`，便于 Beta 发布与归档审计。

## 0.02.09 - 2026-04-08
