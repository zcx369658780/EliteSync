# western_lite 产品承诺边界（2.4）

## 定位

`western_lite` 是 2.4 版本中西占模块的正式运行形态：轻量、受控、低口径、可回退。

## 产品承诺

- 可以作为匹配解释中的辅助模块存在。
- 可以输出可读的展示标签和降级提示。
- 可以参与关系摘要和模块级说明。
- 可以与八字、属相、性格模块一起构成综合解释。
- 可以在不改 API 契约的前提下做 shadow compare 和回滚。

## 不承诺的内容

- 不承诺高精 canonical 星盘生产级准确度。
- 不承诺在部分输入或 fallback 情况下发放高置信 badge。
- 不承诺把西占作为关系判断的唯一主因。
- 不承诺使用 AGPL 受限运行时引擎作为闭源生产默认路径。
- 不承诺将 `western_lite` 解释成“确定性结论”。

## 用户可见措辞

### 推荐
- 轻量参考
- 辅助判断
- 基于当前信息的推断
- 当前结论稳定性有限

### 避免
- 高精
- 确定
- 强结论
- 完全可信
- 绝对匹配

## 运行约束

- 默认模式：`WESTERN_POLICY_MODE=western_lite`
- `engine_mode`、`confidence`、`display_guard` 必须与该模式一致
- 未来若升级到授权 canonical，必须重新出具新的执行决策，不可直接改口径

## 适用范围

- 匹配解释页
- 模块级 insight card
- 周报/简报中的西占描述
- 灰度 / 回滚演练输出

## 一句话承诺

`western_lite` 只提供轻量、可回退、可解释的西占辅助能力，不提供高精生产级确定性承诺。
