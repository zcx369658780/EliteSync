# EliteSync 项目总览

更新时间：2026-04-19

## 项目名称

- `EliteSync`（慢约会）

## 产品定位

- 以“慢约会 / 深度匹配 / 资料真值 / 轻量转化”为核心的社交产品。
- 当前不是单一功能 App，而是一个已经进入版本化治理与持续扩展阶段的完整产品体系。

## 当前开发阶段

- 当前进入 `4.x` 后半段的基础能力补全后阶段。
- 现状不是“搭建主链”，而是“在稳定主链上继续扩展社交能力”。

## 核心差异化能力

- 资料真值优先：`profile/basic` 保存后由服务端重算 canonical truth。
- 星盘/八字/紫微等玄学链路以服务端真值为准，Flutter 只做展示层。
- 非官方四维人格问卷已正式产品化，但不冒充官方 MBTI。
- 图片 / 视频消息已正式接入聊天主链，支持上传、预览、失败重试和摘要回读。
- 版本中心、版本检查、发版脚本和下载包保持严格绑定。

## 当前技术路线概览

- 宿主：Android
- 前端：Flutter module
- 后端：Laravel API
- 发布：阿里云下载包 + 版本检查接口 + release script
- 数据原则：服务端真值优先，前端缓存只兜底

## 当前版本基线

- 对外发布版本：`0.04.04 / 40400`
- 算法版本：`4.4`
- 当前最新已归档版本：`4.4S`

## 一句话结论

- 这个项目已经从“能跑的社交原型”进入“真值链稳定、模块化扩展、版本化治理”的正式产品阶段，并已完成图片 / 视频消息与媒体链稳定性收口。
