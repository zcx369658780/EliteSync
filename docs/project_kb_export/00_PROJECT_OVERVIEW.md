# EliteSync 项目总览

更新时间：2026-05-01

## 项目名称

- `EliteSync`（慢约会）

## 产品定位

- 以“慢约会 / 深度匹配 / 资料真值 / 轻量转化”为核心的社交产品。
- 当前已经从 4.x 的基础能力补全阶段，进入 **4.9 稳定门禁 + 5.x 高价值主链功能覆盖优先** 阶段。

## 当前开发阶段

- `4.9` 已完成验收并冻结为 `pass with observations`。
- 当前主线已切换到 `5.x`：围绕发现、聊天、个人经营页、AI 辅助层、同城与轻关系推进进行高价值主链功能覆盖。

## 核心差异化能力

- 资料真值优先：`profile/basic` 保存后由服务端重算 canonical truth。
- 星盘 / 八字 / 紫微等画像字段以服务端真值为准，前端缓存只兜底。
- 非官方四维人格问卷已正式产品化，但不冒充官方 MBTI。
- 图片 / 视频消息、动态、通知中心、1v1 真语音主链都已形成稳定主干。
- 版本中心、版本检查、发版脚本和下载包保持严格绑定。

## 当前技术路线概览

- 宿主：Android
- 前端：Flutter module
- 后端：Laravel API
- 发布：阿里云下载包 + 版本检查接口 + release script
- 数据原则：服务端真值优先，前端缓存只兜底

## 当前版本基线

- 对外发布版本：`0.04.09 / 40900`
- 算法版本：`4.9`
- 当前 4.9 状态：已完成验收并收口为 `pass with observations`

## 5.x 当前方向

- 发现页：分栏、搜索、同城、轻治理、低压私聊入口
- 聊天页：首聊 / 回聊 / 稍后再聊、关系摘要、AI 续话、关系节奏化语音入口
- 个人页：个人经营区、标签体系、AI 助理、AI 草稿助手、展示标识、外观层
- 设置页：真设置中心与外观层分层，权限前解释优先
- 云端治理：后移到 5.4 / 5.5，当前先服务高价值主链覆盖

## 一句话结论

- 这个项目已经从“能跑的社交原型”进入“4.9 稳定门禁已完成、5.x 开始高价值主链功能覆盖”的正式产品阶段。
