# EliteSync 项目总览

更新时间：2026-05-12

## 项目名称

- `EliteSync`（慢约会）

## 产品定位

- 以“慢约会 / 深度匹配 / 资料真值 / 高质量低频匹配 / 低压社交转化”为核心的社交产品。
- 当前阶段切换为 **6.0 Alpha 内测准备线**：商用级底座重构 + Date Drop 式高质量低频匹配 + 搭子精准陪伴 + 基础社交功能补齐 + 玄学解释产品化 + UI/IA 内测打磨。

## 当前开发阶段

- 5.6-5.10 已完成第一轮“玄学能力二次产品化与校准线”闭环，验收口径为 `pass with observations`。
- 当前不继续直接制定 5.11。
- 当前主计划文件：`docs/version_plans/ELITESYNC_6_0_ALPHA_MASTER_PLAN_2026_05_12.md`。
- 下一步是制定 `v_6_0_A0_Alpha内测准备_商用化底座与路线冻结版_开发计划书_2026_05_12.md`，A0 为 planning-only，不做 runtime。

## 三类参考角色

| 参考对象 | 在 EliteSync 中的角色 | 可吸收重点 | 不吸收内容 |
|---|---|---|---|
| Date Drop | 匹配机制母版 | 低频高质量匹配、深度资料、开放题、反馈驱动、长期关系导向 | 浅层刷卡、无限推荐流 |
| Soul | 社交表达参考 | 交友意图、自我介绍、关系节奏、低压破冰、个人经营 | 强 Feed、关注制、匿名快聊、礼物商城、强运营活动 |
| 测测 / CECE | 玄学解释层参考 | 入口矩阵、档案上下文、字段引导、维度解释、摘要 / 详情分层 | 真人咨询、付费报告、上传截图主路径、娱乐化测试 |

## 当前技术路线概览

- 宿主：Android
- 前端：Flutter module
- 后端：Laravel API，6.0 Alpha 起规划后端 v2 contract-first + parallel migration
- 发布：阿里云下载包 + 版本检查接口 + release script
- 数据原则：服务端真值优先，前端缓存只兜底

## 当前版本基线

- 当前最新对外发布版本仍为：`0.05.05 / 50500`
- 上一条发布基线 / 历史发布链：`0.05.04 / 50400`
- 5.6-5.10 是已完成历史闭环，不作为当前下一步。

## 一句话结论

- EliteSync 已从“5.6-5.10 解释层小步追加”切换为“6.0 Alpha 内测准备线”，先修商用级底座，再重构 Date Drop 式匹配主链，再落地搭子精准陪伴，再补基础社交与玄学解释，最后做 UI/IA 内测打磨。
