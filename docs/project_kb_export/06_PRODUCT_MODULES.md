# 产品模块

更新时间：2026-05-12

## 6.0 Alpha 主模块角色

| 模块 | 6.0 Alpha 角色 | 说明 |
|---|---|---|
| Home | UI/IA 重构对象 | 从功能堆叠改为今日主任务，减少工程感和入口噪声 |
| Match | P1 主链 | 按 Date Drop 式低频高质量匹配重构，强调深度资料、开放题、等待期、揭晓和反馈闭环 |
| Buddy / 搭子 | P1 主线 | 覆盖学习搭子、电影搭子、吃饭搭子、健身搭子等共同兴趣精准陪伴 |
| Chat | P2 社交补齐 | 低压破冰、关系节奏、自我介绍承接和冷场恢复；不得自动代聊 |
| Profile / Me | P2/P3 表达与资料真值入口 | 深度资料、交友意图、自我介绍、个人经营和表达建议；不得让建议反写真值 |
| Astro | P3 解释层 | 星座 / 八字 / 紫微从原材料转为关系解释能力，继续 derived-only / display-only |
| Settings | 门禁与用户控制入口 | 解释层边界、用户控制、隐私与位置展示说明；不做后台治理平台 |

## 模块边界

- Match 不做无限推荐流或浅层刷卡。
- Buddy / 搭子不做泛同城约玩，不弱化安全边界。
- Chat 建议只能辅助表达，不自动发送、不写 `chat_messages`。
- Profile 建议不能自动修改资料。
- Astro 解释不能反写 `profile/basic`、`profile/astro/*` 或 `user_astro_profiles`。
- Settings 不承担后端偏好持久化，除非后续版本有明确 contract。
