# Ziwei Rollback Playbook 2.5

更新时间：2026-03-30

## 触发条件

- 匹配详情页出现解析异常
- 紫微模块导致解释层空白或结构错位
- 紫微分数明显异常
- 回填/影子对拍结果不稳定

## 快速回滚

1. 将紫微权重降为 0 或恢复到保守值
   - `MATCH_ASTRO_WEIGHT_ZIWEI=0`
2. 保留紫微 canonical 数据，但不参与排序
3. 刷新配置缓存
   - `php artisan config:cache`
4. 重新跑匹配契约测试与解释回归
   - `MatchPayloadContractTest`
   - `ZiweiCanonicalServiceTest`

## 结构性回滚

1. 停止紫微命盘写入
2. 保留历史字段，仅恢复读取
3. 关闭匹配层中的紫微模块输出
4. 重新生成回滚说明并同步顾问确认

## 数据安全说明

- 紫微字段为可空 JSON 字段，回滚不要求删除历史数据。
- 允许保留旧画像，只是不再参与排序与展示。

## 回滚后检查

- 匹配结果接口正常返回
- 解释层至少保留八字/属相/星座/星盘/合盘链路
- 前端不应出现灰页或空解释块
