# EliteSync 开发日志（2026-03-21）

## 今日目标
- 修复基础资料与扩展画像的服务端路由/保存问题
- 完成八字扩展算法（大运、流年、五行）并持久化
- 修复登录后画像缓存自动回填
- 统一并重构按钮视觉风格

## 主要完成项

### 1) 后端可用性与路由问题修复（阿里云）
- 诊断并修复 `api/v1/profile/basic`、`api/v1/profile/astro` 异常路径问题。
- 处理 API 鉴权异常：未登录访问返回统一 `401 JSON`，不再抛 `Route [login] not defined`。
- 调整会话配置，避免运行时异常导致接口不可用。

### 2) 基础资料能力补齐
- 性别改为必填二选一（男/女），并接入前后端保存与读取。
- 基础资料新增并打通字段：
  - 昵称（name）
  - 城市（city）
  - 婚恋目标（relationship_goal）
- 城市改为 GPS 自动定位获取（前端）。
- 婚恋目标改为下拉选择：结婚 / 恋爱 / 交友。
- 后端已完成数据库迁移并上线。

### 3) 扩展画像能力增强
- 新增八字扩展计算：
  - 大运（前 8 步）
  - 流年（10 年）
  - 五行图谱（木火土金水计分）
- 计算结果已接入：
  - Android 展示
  - `/api/v1/profile/astro` 保存与读取
  - 服务端持久化字段（JSON）
- 大运/流年改为按性别分支计算（不再默认男命）。

### 4) 登录后自动回填修复
- 登录成功后自动拉取基础资料与扩展画像缓存。
- 若本地输入缺失，优先使用服务端缓存补齐后再进行计算。

### 5) 注册页体验优化
- 密码输入区域新增明确格式提示：
  - 至少 8 位
  - 必须包含字母和数字

### 6) 按钮 UI 重构（最终改为纯色方案）
- 多轮调整后，确认放弃渐变按钮。
- 全部按钮改为纯色状态机（normal/pressed/disabled），统一星空主题配色。
- 同步统一：文本颜色层级、输入框配色、全局字体与字号（Material Typography）。
