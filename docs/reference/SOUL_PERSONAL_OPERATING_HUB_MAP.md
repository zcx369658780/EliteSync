# Soul Personal Operating Hub Map

日期：2026-05-02

## 1. 个人经营页结构

Soul Me 可抽象为：

`空间外观 -> 身份资料 -> 标签表达 -> 资产/商业入口 -> 内容沉淀 -> AI 经营建议 -> 成长激励`

## 2. 模块映射

| 层级 | Soul 表达 | 作用 | EliteSync 参考 |
| --- | --- | --- | --- |
| 空间外观 | 背景、头像、空间图标 | 建立个人空间感 | 个人空间外观预览 |
| 身份资料 | 昵称、IP 属地、关注数 | 基础可信信息 | 资料真值链展示 |
| 标签表达 | ENFJ、职业、恋爱态度 | 快速表达个性 | 标签体系和展示优先级 |
| 展示标识 | 主页佩戴、铭牌、勋章 | 轻身份徽章 | 最多展示少量标识 |
| 内容沉淀 | 瞬间、收藏、赞过 | 个人内容经营 | 状态/动态回流 |
| AI 经营 | 吸引力评分、AI 帮发、诊断 | 降低表达成本 | AI 草稿助手和表达建议 |
| 商业资产 | Soul币、商城、超级星人 | 变现与权益 | 当前不建议照搬 |
