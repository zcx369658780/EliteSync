# Soul Discover Conversion And Governance Map

日期：2026-05-02

## 1. Discover 转化路径

`推荐/同城内容曝光 -> 用户看到身份/距离/在线/图片/正文 -> 私聊 -> Chat 单聊页`

辅助路径：
- `搜索 -> 热榜/AI 话题 -> 内容消费或发帖`
- `同频信号 -> 更多同频 -> 潜在关系推荐`
- `动态更多 -> 私聊/关注/分享`

## 2. 轻治理路径

`动态更多 -> 不喜欢 / 举报 / 复制链接`

设计意义：
- 不喜欢用于推荐质量反馈。
- 举报用于安全治理。
- 复制链接和分享用于外部扩散。
- 私聊和关注与治理同层，但入口样式需防误触。

## 3. EliteSync 可映射项

适合进入 EliteSync：
- Status / Discover 卡片中的低压私聊入口。
- 对每个推荐对象展示一个“为什么值得聊”的理由。
- `不感兴趣 / 举报` 作为轻治理基础。
- 搜索页承载话题建议，而不是只做关键词查找。

不适合当前立即进入：
- 强同城定位授权。
- 大量外部分享渠道。
- 商业化热榜或高频运营浮层。
