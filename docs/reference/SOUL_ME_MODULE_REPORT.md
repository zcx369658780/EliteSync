## 页面概览

当前 Soul 的“自己”页并不是单纯的个人资料卡，而是一个融合了资料展示、内容发布、吸引力评分、AI 辅助发帖、功能中心入口与底部主导航的综合个人中枢。当前可见结构大致分为：

- 顶部个人资料区：头像、昵称、关注数、被关注数、看过我、展示标识、魅力标签、天数、人格标签
- 顶部功能位：切换用户、二维码、设置
- 中部内容区：瞬间 / 收藏 / 赞过、全部、主页吸引力卡、AI 助理卡、个人瞬间内容
- 下部功能区：数字藏馆、Soul 币中心、超级星人、个性商城、娱乐中心
- 底部导航：星球 / 广场 / 发布瞬间 / 聊天 / 自己

个人页的表达方式偏“画像化 + 吸引力评分 + AI 生成建议”，并不只是简单的资料查看页。

## 对 EliteSync 的参考价值

- 个人页不是单纯资料页，而是“资料展示 + AI 赋能 + 内容发布 + 功能中心”的混合中枢。
- 标签区域很关键，用户通过 `人格 / 职业状态 / 情感态度` 等标签来表达自己。
- `查看AI助理` 这种入口很适合做成 EliteSync 里的内容辅助、关系推进或主页建议卡。
- `主页吸引力评分` 很适合作为 EliteSync 里的轻量健康度 / 活跃度 / 内容质量提示。
- 功能区采用大图标宫格，适合放二级入口，但不抢主资料卡的视觉权重。
