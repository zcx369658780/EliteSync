## 必测功能

### 资料与画像
- 可修改出生资料
- 修改后会触发服务端重算
- summary / chart 可刷新
- 出生时间 / 出生地点 / 经纬度展示一致

### 星盘显示
- 本地绘制正常
- 盘面元素开关可生效
- 恢复默认可生效
- 预设档位可切换
- 图层显示与说明区不互相遮挡

### 设置中心
- 设置首页结构清楚
- 盘面设置入口可达
- 隐私 / 安全 / 通知 / 性能入口可达
- 返回栈正确

### 关于与更新
- 版本号展示正确
- 更新历史展示正确
- 下载信息展示正确
- 服务状态说明不误导

### Admin 与口径
- Admin 用户列表可识别测试账号 / 合成用户 / 普通用户
- 统计卡片解释正确
- 版本与发布口径一致

## 手工验收步骤

1. 修改一个出生资料字段，确认页面提示与数据刷新正常
2. 打开星盘主页面，确认盘面完整且元素开关有效
3. 切换到预设档位，确认显示变化真实发生
4. 打开设置中心，确认各入口层级清楚
5. 打开关于与更新页，确认版本与更新说明一致
6. 打开 Admin 用户列表，确认统计口径说明不再歧义
7. 截取关键页面作为验收证据

## 失败后补测

- 资料保存失败：先查接口返回与重算链路
- 星盘不刷新：先查本地偏好与 summary/chart 数据
- 设置入口丢失：先查路由与返回栈
- 版本不一致：先查 about / app_update / changelog / release log
