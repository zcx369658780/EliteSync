# 4.7 截图验收说明

更新时间：2026-04-26

## 验收目的

本说明用于证明 4.7 的现代 UI 基线已恢复并受保护，避免后续跨层恢复把页面再退回旧版。

## 证据范围

当前已收集并索引的主要页面证据包括：

- 首页
- 消息列表
- 聊天页
- 通知中心
- 匹配页
- 状态流 / 发布状态页
- 我的 / 设置 / 退出登录入口
- 版本中心

对应索引见：

- [`docs/version_plans/4.7_UI_BASELINE_EVIDENCE_INDEX.md`](D:/EliteSync/docs/version_plans/4.7_UI_BASELINE_EVIDENCE_INDEX.md)

## 通过标准

截图验收时，至少应确认：

1. 底部 Tab 保持现代布局
2. 首页保留现代卡片与入口结构
3. 消息页仍保留聊天入口
4. 聊天页仍保留通话入口
5. 通知中心仍是现代卡片风格
6. 匹配页仍保留现代说明卡片
7. 我的 / 设置 / 退出登录入口仍可发现
8. 版本中心仍可观测版本、服务状态与更新检查

## 当前状态

当前证据已经足够支撑 4.7 的 UI baseline 复验。
