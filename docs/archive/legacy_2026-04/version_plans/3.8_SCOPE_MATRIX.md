# EliteSync 3.8 范围矩阵

更新时间：2026-04-16  
版本状态：stage 0 拆解

## 1. 范围总览

3.8 的目标不是扩成全量专业占星平台，而是先把“专业工作台深化首版”落成。

## 2. P0 / P1 / 暂缓项

### 2.1 P0

- 黄道制切换
- 宫位制扩展
- 相位密度 / 相位集合
- 容许度
- 点位范围 / 点位集
- 参数解读弹窗
- 星体解读
- 相位解读
- 宫主星解读
- 飞宫解读
- 主星落宫摘要
- 宫主星落宫摘要
- 宫主飞宫摘要
- 最小相位表摘要
- 校准样例集
- 校准报告
- 视觉与交互回归

### 2.2 P1

- 更多宫位制
- 第一批小行星 / 虚点解释
- 更完整的相位表视图
- 桌面 / 宽屏布局预演
- 大限 / 流年解释骨架

### 2.3 暂缓项

- 全量法达 / 小限 / 大限 / 流年体系
- 全量三限 / 次限 / 推运专业平台化
- canonical 引擎全面替换
- 登录 / 会话 / 地理编码 / 数据库大改
- 全量专业桌面软件级参数海洋

## 4. 旧入口保护

必须继续保持可达：

- 资料主页 / 编辑资料页
- 星盘主页面
- 盘面设置页
- 路线工作台
- 设置中心
- 版本中心
- 高级演示页
- 匹配 / 首聊 / 广场 / synthetic / 治理链路

## 5. 3.8 对外口径

3.8 对外只能说：

> EliteSync 已进入专业工作台深化阶段，新增参数与解读一体化首版，并开始把参考样例纳入正式校准闭环。

不得对外宣称：

- 已成为全量专业占星平台
- 已支持所有限运体系
- 已替换 canonical 引擎
