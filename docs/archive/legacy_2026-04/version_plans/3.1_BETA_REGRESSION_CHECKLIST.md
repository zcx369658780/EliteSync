# EliteSync 3.1 Beta 回归清单

更新时间：2026-04-09

## 最小 smoke

1. 登录成功
2. 测试账号池可读取
3. 管理员入口可见且仅管理员可见
4. 发布状态入口可见
5. 广场页可加载

## 必测功能

### 测试账号池
- 可生成
- 可重建
- 生成后能读到 `account_type=test`
- 画像补齐完成
- 能参与匹配或按配置被排除

### 管理员控制
- 查看用户列表
- 封禁 / 解封账号
- 发起每周匹配
- 删除状态
- 查看匹配批次 / 状态信息

### 状态发布
- 普通用户可发状态
- 测试账号可发状态
- 状态可进入广场
- 删除后不再出现在广场

### 广场可见性
- 普通账号能刷到状态
- 测试账号按配置可见 / 可隔离
- 指标隔离字段生效

### 匹配调试
- 测试账号可参与匹配
- 管理员可触发匹配
- 触发后批次可回查

## 回归检查

- 注册 / 登录
- profile/basic
- astro summary / chart
- match current / history
- messages / 首聊
- about / version / health

## 失败后补测

- 数据写入失败：先查脚本、字段、默认值
- 权限失败：先查管理员边界
- 广场不显示：先查可见性字段与 feed 读取
- 匹配失败：先查 debug 开关与资格字段
