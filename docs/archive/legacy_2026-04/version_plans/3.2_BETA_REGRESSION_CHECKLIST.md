## 手工验收步骤

1. 创建一批 synthetic 账号，确认 batch_id、数量、seed、状态记录完整
2. 抽查 2-3 个 synthetic 账号，确认基础资料、出生地、八字、星盘可用
3. 进入 admin dashboard，确认批次/状态概览可见
4. 进入 admin users，确认可筛选、可封禁、可识别账号类型
5. 进入 admin verification，确认审核队列与动作可用
6. 进入状态广场，确认普通账号 A 发布状态后，普通账号 B 可刷到
7. 验证首页广场预览与广场页内容一致
8. 验证匹配后首聊入口、破冰建议、恢复建议可用
9. 验证八字 / 星盘 summary 与 chart 页面显示稳定、无重叠
10. 执行 batch cleanup / disable，确认数据回收和可见性收口
11. 截取关键页面与操作结果，作为验收证据
