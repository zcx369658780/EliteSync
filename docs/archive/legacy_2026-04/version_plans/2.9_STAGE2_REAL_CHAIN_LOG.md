- 本轮关于页最初显示旧 UI 的根因不是源码未更新，而是验收时误装了 `apps/flutter_elitesync_module` 的独立产物。
- 真机 / 模拟器验收 Flutter 页面时，必须优先使用 `apps/android` 构建出的 host APK（`com.elitesync`），并在必要时先 `adb uninstall com.elitesync` 再重装。
