## 2. 3.9a 的完成内容

### 2.1 UI 文案降噪

- 高级时法预览页的用户可见文案已从工程化口径收敛为更自然的展示口径
- 高级时法演示页、设置页参数联动、高级时法框架卡都已同步收口
- `scaffold`、`advanced-context` 等工程词汇已从主要用户可见说明中压低

### 2.2 细粒度解释层证据补强

- 新增了显式的 `证据 -> 解释 -> 提示` 组织说明
- 细粒度解释层可以直接用于截图、复核和归档
- 解释层仍保持 `display-only` 边界，不回写 canonical truth

### 2.3 回归材料加厚

- 回归报告已升级为执行记录
- 截图证据索引与验收说明已加入回归映射
- 旧入口、设置页、预览页、演示页、版本中心的可达性仍在保护范围内

### 2.4 版本边界收口

- `pair_chart / timing / advanced-context` 已补齐边界说明
- 3.9C 只做说明层收口，不改运行时契约
- `canonical truth`、本地展示偏好、路由常量和 schema 均未改写
