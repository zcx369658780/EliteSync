# EliteSync 3.8 执行计划

更新时间：2026-04-16  
版本状态：stage 5 final regression prep 版  
目标版本：3.8

## 1. 基本信息

- 需求名称：3.8 专业工作台深化版
- 需求来源：`elite_sync_3_8_版本开发计划书_2026_04_13.md`
- 计划阶段：stage 0
- 负责人：Codex
- 协作角色：Claude-MCP / Gemini-MCP
- 当前稳定基线：`3.7` 已正式归档，`0.03.07 / 30700` 为当前发布版本

## 2. 需求理解

- 用户要的是把现有星盘能力从 `3.7` 的高级预览与归档版，推进到“参数 + 排盘 + 解读”一体化的专业工作台首版。
- 需要补齐的不是单一页面，而是一个可操作、可阅读、可校准、可归档的参数工作台闭环。
- 这版不能碰 canonical truth 大改，不能把派生解释层伪装成新真值层。
- 这版也不能把移动端做成参数过载的后台，要保持信息层级清楚、旧入口可达、旧缓存可读。

## 3. 本版不改什么

- 不做 canonical 引擎全面替换
- 不改登录 / 会话 / 路由 / 地理编码 / 数据库大结构
- 不把参数扩成全量专业软件级控制台
- 不删旧字段、不破坏 `chart_data` 兼容读取
- 不破坏 3.7 高级演示页、路线工作台和版本中心

## 4. 受影响模块

- 参数工作台：黄道制、宫位制、相位密度、容许度、点位范围
- 解读层：参数解读、星体解读、相位解读、主星解读、飞宫解读
- 校准层：参考样例集、校准报告、已知偏差
- 视觉层：本命盘页、参数页、解读卡、日期选择器
- 文档层：执行计划、风险评审、范围矩阵、差距矩阵、校准报告、截图索引
