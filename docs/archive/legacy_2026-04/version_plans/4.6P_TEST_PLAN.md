# 4.6P 测试计划

更新时间：2026-04-24

## 最小测试集

### 后端

- `GET /api/v1/rtc/calls/{callId}/livekit` 返回值与 room / identity / token 一致
- heartbeat 不会过早误杀正常通话
- 断连后会话能正确收口，不残留 busy

### Flutter

- 真机与模拟器都能进入通话页
- `TrackPublished` / `TrackSubscribed` / `remoteAudioTrack` 日志能打出来
- 音频频谱能显示真实帧或明确等待原因
- 播放链能记录 attach / play / route 相关诊断

### 回归

- RTC 状态机原有测试继续通过
- 消息 / 通知 / 图片 / 视频 / 问卷 / 动态主链不受影响

## 手工验收

- 真机发起 -> 模拟器接听
- 模拟器发起 -> 真机接听
- 记录双方是否真正听到声音
- 若仍无声，明确失败点是 publish / subscribe / playback / routing 中的哪一步
