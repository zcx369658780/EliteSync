## Baseline quality requirement

- 至少一张手机、一张模拟器
- 至少一张首页基线
- 至少一张消息页入口
- 至少一张聊天页入口
- 至少一张设置 / 退出登录
- 至少一张通知中心
- 至少一张版本中心
- 通话页必须能证明入口存在，而不是只有 call ended 回放页
