## 2. 3.5 P0 详细范围

1. 通过 additive change 补齐 engine / schema 元信息。
2. 建立最小参数面板。
3. 升级盘面可读性。
4. 提供参数最小解释。
5. 固定样例对照与回归清单。

---

## 3. 3.5 P1 详细范围

1. 更多宫位制。
2. 更多点位 / 敏感点 / 小行星。
3. 更细的相位层级。
4. 更好的参数信息层级。
5. 参数结果摘要页。

---

## 4. 明确不纳入

1. 合盘、男女对比盘。
2. 行运、三限、次限、月返、日返。
3. 法达、小限、大限、流年。
4. 完整古典 / 现代双路线一次性上线。
5. 高风险 canonical 引擎替换。
