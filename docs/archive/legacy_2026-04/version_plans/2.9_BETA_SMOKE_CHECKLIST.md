# EliteSync 2.9 Beta Smoke Checklist

更新时间：2026-04-08

## 1. 目标

确认 Beta 构建在真实设备 / 模拟器上可启动、可登录、可进入关键页面。

## 2. 必跑命令

- `scripts/smoke_backend_alpha.ps1`
- `scripts/release_gate_alpha.ps1`
- `scripts/regression_alpha_baseline.ps1`

## 3. 最小烟测项

- 登录成功
- `/api/v1/app/health` 返回正常
- 首页首屏可见，不白屏
- 资料页可见
- 匹配页可见
- 消息页可见
- 关于页版本与健康状态可见

## 4. 失败处理

- 先确认是否安装了 `apps/android` host APK
- 再确认当前账号是否有效
- 再确认后端健康检查与版本检查是否正常
- 最后再看是否是慢网超时兜底行为
