# EliteSync 3.7 范围矩阵

更新时间：2026-04-12  
版本状态：stage 0 拆解

## 1. 范围总览

3.7 的目标不是扩展所有高级占星功能，而是先把以下第一批能力做成受控首版：

- 合盘（Synastry）
- 对比盘（Bi-wheel / Comparison）
- 行运（Transit）
- 返照中的一项最小能力（月返优先，必要时可降为日返）

## 2. P0 / P1 / 暂缓项

### 2.1 P0

- 合盘最小入口
- 对比盘最小入口
- 行运最小入口
- 月返或日返最小入口
- 高级解释卡
- 高级样例矩阵
- 差异报告
- 已知偏差归档

### 2.2 P1

- 月返与日返双支持
- 更细的关系摘要
- 更完整的时间窗口筛选
- 桌面 / 大屏高级工作台预演
- 第一批限运原型入口

### 2.3 暂缓项

- 法达 / 小限 / 大限 / 流年全量体系
- 三限 / 次限全量开放
- 全量事件预测软件级参数工作台
- canonical 引擎全面替换
- 登录 / 会话 / 地理编码 / 数据库大结构重构

## 3. 受控能力边界

| 能力 | 是否进入 3.7 P0 | 数据角色 | 接口角色 | 页面角色 |
|---|---|---|---|---|
| 合盘 | 是 | derived-only | 可增量扩展 | 内测入口 / 页面卡 |
| 对比盘 | 是 | derived-only | 可增量扩展 | 内测入口 / 页面卡 |
| 行运 | 是 | derived-only | 可增量扩展 | 内测入口 / 页面卡 |
| 返照（月返 / 日返） | 是 | derived-only | 可增量扩展 | 内测入口 / 页面卡 |
| 限运全量 | 否 | 暂缓 | 暂缓 | 暂缓 |

## 4. 旧入口保护

必须继续保持可达：

- 资料主页 / 编辑资料页
- 星盘主页面
- 盘面设置页
- 设置中心
- 版本中心
- 路线模板与 3.6 差异报告
- 匹配 / 首聊 / 广场 / synthetic / 治理链路

## 5. 3.7 对外口径

3.7 对外口径仅能说：

> EliteSync 已进入推运与高级解读首版，新增合盘 / 对比盘 / 行运 / 最小返照入口，并保持 canonical truth 单源和旧入口兼容。

不得对外宣称：

- 已成为全量专业占星平台
- 已支持所有限运体系
- 已替换 canonical 引擎
