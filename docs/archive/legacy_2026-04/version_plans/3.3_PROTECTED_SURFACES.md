# EliteSync 3.3 保护面清单

更新时间：2026-04-11

## 必须保护的主链路

### 1. 注册 / 登录
- `POST /api/v1/auth/register`
- `POST /api/v1/auth/login`
- 登录态恢复

### 2. 资料与画像
- `POST /api/v1/profile/basic`
- `GET /api/v1/profile/astro/summary`
- `GET /api/v1/profile/astro/chart`
- `GET /api/v1/geo/places`

### 3. 星盘显示
- 星盘主页面
- 星盘设置页
- 本地 SVG 绘制
- 恢复默认 / 预设档位
- 元素开关

### 4. 设置中心
- 设置首页
- 账号安全页
- 隐私设置页
- 通知设置页
- 性能偏好页
- 盘面设置页

### 5. 关于与更新
- 关于页
- 更新历史页
- 版本检查页
- 下载地址展示

### 6. 运营与治理
- `GET /api/v1/admin/*`
- `GET /admin/users`
- `GET /admin/dashboard`
- `GET /admin/verification`
- Admin 用户统计卡片

### 7. 版本与服务状态
- `GET /api/v1/app/health`
- `GET /api/v1/app/version/check`
- release log
- changelog

## 新增应重点保护的 surfaces

### 后端
- 出生资料更新后的重算链路
- `UserAstroMirrorService`
- `AstroProfileController`
- 版本检查与下载配置

### 前端
- 星盘设置页的显示偏好状态
- 设置中心入口与返回栈
- 关于与更新页的版本说明
- 资料编辑页的地点搜索与提交按钮

## 明确不动的边界

- 不把本版改成星盘算法重写版
- 不把盘面设置页扩成专业占星调参后台
- 不把本地偏好回写成 canonical 真值
- 不破坏 3.2 / 3.2a 已通过的星盘显示与账号体系能力
