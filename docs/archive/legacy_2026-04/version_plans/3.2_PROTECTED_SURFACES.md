# EliteSync 3.2 保护面清单

更新时间：2026-04-10

## 必须保护

- 注册 / 登录
- 基础资料保存
- 出生地搜索
- 画像重算
- 匹配解释
- 每周匹配
- 首聊 / 会话
- 状态发布 / 广场显示
- 管理员移动端控制面
- synthetic batch 创建与清理
- 3.1 已通过能力

## 保护原则

1. canonical 数据只能由服务端决定
2. 前端缓存不能成为第二真源
3. synthetic/test/normal 必须严格分层
4. synthetic 默认 `exclude_from_metrics = true`
5. 玄学计算层、解释层、展示层必须分离
6. 任何批次 / 迁移 / 配置改动都要有回滚点
7. 不得为了 3.2 顺手重写无关业务链路
8. 不得让临时验收数据长期污染环境
9. 管理员控制面必须与普通用户链路隔离
