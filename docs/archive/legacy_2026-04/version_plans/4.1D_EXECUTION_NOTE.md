## Scope

This note records the lightweight linkage work for 4.1:

- questionaire summary snapshot persistence
- home page summary card
- match result summary card
- first-chat helper prompt linkage
- minimal observability for summary card actions and questionnaire entry / submit / result / history / retest events

## Implemented

### Questionnaire summary snapshot

- Added `questionnaire_profile_snapshot_v1` in local cache.
- Persisted the latest questionnaire summary after successful submission.
- Exposed a `questionnaireProfileSnapshotProvider` for lightweight reuse.

### Home page linkage

- Added a questionnaire summary card below the shortcut row.
- The card links to:
  - questionnaire history
  - questionnaire continuation / retest
- Added telemetry for both actions.

### Match result linkage

- Added the same questionnaire summary card to match result.
- First-chat suggestions now take the questionnaire summary into account when available.
- Added telemetry for both summary-card actions.

### Questionnaire observability

- Added telemetry for:
  - questionnaire entry opened
  - questionnaire submitted
  - questionnaire result viewed
  - questionnaire history opened
  - questionnaire retest started
  - homepage personality summary opened
  - match personality hint opened
