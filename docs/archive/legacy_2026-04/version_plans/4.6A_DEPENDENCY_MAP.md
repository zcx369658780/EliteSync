## 隐藏耦合点

- 会话页与通知中心的回流语义
- 通话结果是否影响消息摘要
- 失败态是否误写为 canonical truth
- 本地缓存是否污染通话状态
- 旧 websocket gateway 是否需要复用或并行

## 不可擅自改动的边界

- `chat_messages` 文本 / 图片 / 视频主链
- `notifications` 已归档闭环
- `media_assets` / `message_attachments`
- `profile/basic`、`astro/summary`、`astro/chart`
- `app/health`、`app/version/check`
