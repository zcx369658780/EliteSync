# 4.8 截图验收说明

更新时间：2026-04-28

## 验收目标

本页用于说明 4.8 UI baseline 与 RTC smoke 的截图证据如何理解与验收。

## 已纳入证据的页面

- 首页
- 消息列表
- 聊天页
- 通知中心
- 匹配页
- 我的页
- 设置页
- 版本中心
- RTC 通话页
- RTC 音频帧现场证据

## 验收原则

1. 截图用于证明现代 UI 基线仍在，不用于冒充功能完成。
2. 通话页是否连通，以现场联测与 call state / audio frame 证据为主。
3. 若截图时机未完整捕捉到音频帧，优先信任现场联测确认。
4. 通知中心里的工程 slug 当前仍属于 observation，不作为 4.8 阻断项。
