# EliteSync 3.4 Beta 回归清单

更新时间：2026-04-11  
对应版本：3.4

## 1. 最小 Smoke

- 启动 App
- 登录
- 打开设置中心
- 打开资料编辑页
- 打开星盘主页面
- 打开盘面设置页
- 打开版本中心页
- 打开隐私设置页

## 2. 资料与真值链路

- 修改出生时间并保存
- 修改出生地点并保存
- 保存后确认页面有明确反馈
- 保存后确认服务端重算完成
- 返回资料页 / 星盘页确认内容刷新
- 慢网时确认不会长时间白屏
- 保存失败时确认有明确提示

## 3. 星盘显示回归

- 打开本命盘页确认本地绘制正常
- 切换预设档位确认盘面变化
- 打开恢复默认确认盘面回到推荐状态
- 开关图层后确认显示层变化生效
- 确认盘心信息、标签、相位线没有异常裁切

## 4. 设置中心回归

- 进入设置中心确认分组结构正常
- 进入盘面设置页确认说明文案清楚
- 进入性能偏好页确认仅影响本地偏好
- 进入隐私设置页确认可见性边界说明清楚
- 进入版本中心页确认版本与服务状态可读

## 5. 版本与可观测性

- `GET /api/v1/app/health` 返回正常
- `GET /api/v1/app/version/check` 返回正确版本判断
- about / 版本中心页标题与索引口径一致
- 更新历史标题与当前版本口径一致

## 6. Beta 运营准备

- 弱网首进验证
- 回滚检查
- 关键页面截图留档
- smoke / regression / acceptance 记录归档

## 7. 失败后的补测建议

- 先重跑 `flutter test test/features/acceptance_smoke_test.dart`
- 再重跑 `flutter analyze --no-fatal-infos`
- 后端重跑 `php artisan test --filter=ProfileBasicRecomputeTest`
- 后端重跑 `php artisan test --filter=AppVersionApiTest`
