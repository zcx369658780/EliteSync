# 3.9 UI 信息架构说明

更新时间：2026-04-17

## 页面分层

- 设置页：参数入口与本地偏好
- 盘面设置页：路线 / 工作台 / 参数解读
- 高级时法预览页：时法框架 + 路线对照 + 样例矩阵
- 高级时法演示页：离线归档与视觉验收
- 细粒度解释层卡：摘要层 / 条目层 / 高级时法关联层

## 信息顺序

1. 先看路线与工作台参数
2. 再看高级时法框架
3. 再看关系维度样例
4. 最后看日志与已知偏差

## 命名边界

- “参数”仍指本地展示与工作台口径
- “高级时法”指 advanced-context scaffold
- 不使用“真值”字样描述时法 scaffold
