# 4.2 执行记录

更新时间：2026-04-18

## 版本定位

`4.2` 是 `图片消息正式接入版`。这版的目标不是再造媒体平台，而是把 4.0 已完成的媒体基础设施正式接到聊天主链上，形成可发送、可接收、可预览、可失败重试、可缓存、可回归的图片消息能力。

## 当前已完成内容

### 后端

- `POST /api/v1/messages` 已支持 `attachment_ids`
- 图片附件可绑定到 `chat_messages`
- `GET /api/v1/messages` 和会话摘要均可回读图片消息
- 会话摘要会把纯图片消息概括为 `图片消息`
- 最小 telemetry 事件已放行：
  - `chat_image_picker_opened`
  - `chat_image_upload_started`
  - `chat_image_upload_succeeded`
  - `chat_image_upload_failed`
  - `chat_image_message_sent`

### Flutter

- 聊天页已支持选图、上传、失败重试、重新选择与清除
- 图片消息可绑定 `attachmentIds` 并发送到后端
- 图片气泡已可渲染并支持预览
- 附件面板与本地草稿态已经接入聊天页

### 归档收口

- walkthrough 级截图证据已补齐并保留在仓库根目录
- 顾问交接稿、closeout note、截图索引与验证说明已升级为最终版
- debug 宿主已重新构建并安装，用于产出当前证据包
