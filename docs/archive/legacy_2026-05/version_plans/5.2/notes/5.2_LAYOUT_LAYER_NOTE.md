# EliteSync 5.2 Layout Layer Note

## Goal

Add a lightweight personal appearance layer without turning settings into a commercial dressing center.

## Implementation

`ProfilePage` now includes `个人空间外观` with:

- `展示封面：温柔清爽`
- Style chips: `清爽`, `安静`, `真诚`
- Copy that separates appearance from privacy, account, and security settings

## Boundaries

- No shop
- No coin system
- No paid dressing rights
- No settings route replacement
- No profile truth write-back

## Observation

The current layer is a preview and semantic slot only. Persistent appearance preferences should be designed separately if needed.
