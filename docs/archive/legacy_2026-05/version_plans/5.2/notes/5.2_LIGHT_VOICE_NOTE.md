# EliteSync 5.2 Light Voice Note

## Goal

Add a light voice expression candidate slot without touching RTC or LiveKit.

## Implementation

`ProfilePage` now includes `轻语音表达` with:

- `声音名片候选位`
- `录制前说明`
- A dialog explaining that this is a candidate slot and real recording would request microphone permission only when needed

## Boundaries

- No RTC state machine change
- No LiveKit token, join, leave, heartbeat, or permission service change
- No multiplayer voice, video, live, or audio social platform
- No recording persistence in this slice

## Observation

The current slice is a candidate position and pre-recording explanation. It proves product semantics without introducing an audio contract.
