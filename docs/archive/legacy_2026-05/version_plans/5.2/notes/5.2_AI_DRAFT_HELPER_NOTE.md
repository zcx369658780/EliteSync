# EliteSync 5.2 AI Draft Helper Note

## Goal

Add a minimum AI expression helper for self-introduction, greeting, and highlight organization.

## Implementation

`ProfilePage` now includes `AI 草稿助手` with three local draft actions:

- `帮我写一句自我介绍`
- `帮我写一句问候`
- `整理我的亮点`

Tapping each action opens a bottom sheet showing a draft suggestion and a disclaimer.

## Boundaries

- Drafts are local presentation suggestions only
- No automatic publishing
- No automatic message sending
- No server truth write-back
- No server-side AI queue

## Observation

The helper does not yet persist drafts. That is intentional for 5.2 because persistence would require a separate product and contract decision.
