## 当前状态

本轮已补齐 RTC 体验层的两个最小壳页：

- `RtcIncomingCallPage`：来电页
- `RtcCallResultPage`：通话结果页

这两页并没有扩成完整 RTC 平台，只是把 `4.6B` 的最小闭环继续补成更清晰的用户路径：

1. 来电通知可进入来电页
2. 来电页可接听 / 拒绝
3. 接听后进入通话页
4. 拒绝或终态可进入结果页
