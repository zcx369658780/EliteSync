# 3.9 时法 Schema 说明

更新时间：2026-04-17

## 设计原则

- 只做挂接式扩展
- 不污染 natal 主结构
- 不回写 canonical truth
- 所有新增字段都应能被解释为 display-only 或 advanced-context

## 当前骨架

- `AstroTimingFrameworkBundle`
- `AstroTimingSignal`
- `AstroTimingSampleCase`
- `AstroTimingMode`
- `AstroExplainabilityEntry`

## 细粒度解释层

- 摘要层只保留每个预览项的总览信息
- 条目层拆分为相位级、点位级、主星 / 宫位关联级三类入口
- 高级时法关联层只承接 advanced-context，不回写 natal 主结构

## 约束

- 年度视角可演示、可审查、可归档
- 主时段视角当前先保留占位能力
- 后续若扩展更多时法，仍需保持 additive change
