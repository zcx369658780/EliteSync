# 4.4 交接说明

更新时间：2026-04-19

## 这版交付了什么

- 聊天主链中的单视频消息最小闭环。
- 视频附件绑定到 `chat_messages` 的主路径。
- 视频消息的上传、成功态、失败重试、状态展示、预览 / 播放入口。
- 会话列表摘要已切换为 `视频消息` 语义。
- 兼容图片消息与文本消息共存。
- 视频相关 telemetry 最小闭环。

## 当前可直接继承的页面与能力

- 聊天页：图片 / 视频附件入口、待发送态、失败态、重试态。
- 消息气泡：图片与视频附件的区分渲染。
- 视频预览页：基础播放 / 返回。
- 会话列表与摘要：可回读视频消息语义。

## 保护面

- `chat_messages` 文本主链
- `media_assets` / `media_processing_jobs` / `message_attachments`
- `profile/basic`
- `profile/astro/summary`
- `profile/astro/chart`
- `app/health`
- `app/version/check`
- `4.0` 媒体底座
- `4.1` 人格问卷
- `4.2` 图片消息主链
- `4.3` 动态流基础版
- `3.x` 已归档真值链

## 明确没有纳入本版的内容

- 视频动态 / 视频广场
- RTC / 通话 / 在线状态系统
- 视频剪辑、滤镜、配乐
- 工业级分片上传完整版
- 大型媒体平台化改造

## 后续承接建议

- 如果继续扩视频能力，先写新计划书，不要回头改 `4.4`。
- 后续版本优先围绕治理、稳定性、播放体验等增量能力开展。
- 不要再把视频摘要回退成图片摘要或正文优先摘要。
