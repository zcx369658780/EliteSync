## 已完成内容

### 4.2A 后端契约与附件绑定

- `POST /api/v1/messages` 已支持 `attachment_ids`
- 图片附件可绑定到消息主链
- `GET /api/v1/messages` 和会话摘要均可回读图片消息

### 4.2B 选图、上传、发送闭环

- Flutter 聊天页支持图片选择和上传
- 上传成功后可发送带附件的消息
- 上传失败、重选、清除与本地草稿态都已具备

### 4.2C 图片渲染与预览

- 图片气泡可以渲染
- 点击后可打开预览页

### 4.2D 最小治理与弱网口径

- 已保留 mime / size / status 的基础治理语义
- 已保留失败态与重试语义

### 4.2E 最小 observability

- 已有图片相关 telemetry 事件
- 触发链可在本地测试与代码层验证
