## 1. 证据范围

3.6 的截图证据只需覆盖路由工作台、路线解释层、星盘详情页和版本中心即可。

## 2. 建议截图列表

1. `3_6_settings_route.png`
   - 设置页中的路线模板与路线差异报告
2. `3_6_natal_route_card.png`
   - 本命盘详情页中的 compact 路线解释卡，或等效的路线说明卡截屏
3. `3_6_natal_chart.png`
   - 本命盘详情页主图与阅读提示
4. `3_6_route_capability_matrix.png`
   - 路线能力矩阵或等效导出截图
5. `3_6_route_parity_report.png`
   - 路线差异报告的展开视图
6. `3_6_multi_route_sample_set.png`
   - 多路线样例集或等效导出截图
7. `3_6_version_center.png`
   - 版本中心 / 关于与更新页

## 3. 说明

- 本索引只记录归档需要的关键页面，不要求覆盖所有设置子项。
- 路线解释层截图需明确显示 `derived-only / display-only` 或等效语义。
- 多路线样例集截图需显示至少 baseline / dense-modern 两组样例。
