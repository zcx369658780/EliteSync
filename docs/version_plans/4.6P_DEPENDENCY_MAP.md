## 关键依赖

- LiveKit join-info 生成
- room.connect / heartbeat / disconnect
- remoteAudioTrack 绑定与播放链
- Android 音频路由与 speaker / earpiece 设置
- 真机 / 模拟器双向复验

## 隐藏耦合点

- 当前音频频谱卡片只是诊断层，不是播放 sink
- `remoteAudioTrack` 的赋值来源与页面 rebuild 之间可能存在生命周期耦合
- 心跳、结果页与通话页面之间的路由收口会影响 busy / timeout 判定
- 真机与模拟器音频输出路由可能不同

## 不可擅自改动的边界

- 不改 4.0 / 4.1 / 4.2 / 4.3 / 4.4 / 4.5 主链
- 不重做 RTC 状态机
- 不把播放链问题扩成多人 / 直播 / 视频版
- 不重建数据库 schema
