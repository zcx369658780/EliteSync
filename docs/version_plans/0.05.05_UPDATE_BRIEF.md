# EliteSync 0.05.05 发布短报

更新时间：2026-05-08

## 1. 发布定位

- VersionName：`0.05.05`
- VersionCode：`50500`
- 版本线：`5.5` 真实小样本反馈吸收版
- 上一版发布包保留：`0.05.04 / 50400`

## 2. 本次包含范围

本次发布打包的是已批准的 5.5 范围：

- 状态广场与状态作者页兼容 Map / List 形态响应，修复相关类型转换问题。
- 个人页浮动 `发布状态` 入口作用域修正，避免在非首页场景遮挡内容。
- 玄学总览页紫微预览裁切修正。
- RTC 接收端来电 watcher 修复。
- RTC 终态来电页 UI 修正，已结束通话不再展示 `接听` / `拒绝`。
- Claude Appium 测试员流程引入与 5.5 验收证据收口。
- 双端 RTC 证据、阿里云 30 秒 invite timeout 部署一致性，以及写入式 RTC 复测证据。

## 3. 保护边界

本次发布不包含：

- matching algo marker 语义变更。
- 数据库 schema / migration / restore 执行。
- 后端重写。
- 覆盖既有 `0.05.04 / 50400` 发布包。
- 超出已批准真实小样本反馈范围的 5.5 runtime 扩张。

## 4. 仍保留的 Observation

- restore drill 仍未执行。
- matching marker 语义仍继续排除。
- 远端 debug / local 环境硬化仍是独立运维项。
- 既有 Flutter info 级 lint 清理仍作为独立任务处理。
- Match Detail 成功态仍需要真实活跃匹配数据复测。

- 当前对外发布线已从 `0.05.04 / 50400` 推进到 `0.05.05 / 50500`。
