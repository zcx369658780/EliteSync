# EliteSync 2.9 Environment Matrix

更新时间：2026-04-08

## 1. 环境职责

### Local

- 日常开发
- 模拟器截图
- 单测 / 调试

### Dev

- 模拟真实联调
- Dev-only mock / debug 登录
- 后台验收入口

### Preprod

- 真链路联调
- smoke / regression / release gate 验证
- 灰度前演练

### Prod

- 真实用户
- canonical 真值
- 只允许稳定配置与受控发布

## 2. 关键配置

- `ELITESYNC_INITIAL_ROUTE`
- `ELITESYNC_ADMIN_MOCK`
- `ELITESYNC_DEBUG_AUTO_LOGIN_PHONE`
- `ELITESYNC_DEBUG_AUTO_LOGIN_PASSWORD`
- `app_update.php`
- `appThemeMode`

## 3. 环境纪律

- dev / preprod 的 mock 不得污染 prod
- host APK 是验收基准，不以 module 产物代替
- 任何发布前必须明确当前环境
