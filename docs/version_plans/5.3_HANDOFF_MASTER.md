## 3. Implemented Runtime Scope

### 3.1 Discover

Changed runtime files:

- `apps/flutter_elitesync_module/lib/features/discover/presentation/pages/discover_page.dart`
- `apps/flutter_elitesync_module/test/features/discover/presentation/pages/discover_page_test.dart`

What changed:

- Added a long-press action sheet on content cards.
- The sheet shows a local recommendation explanation, not a server truth rewrite.
- The card actions support:
  - `继续看详情`
  - `稍后再看`
  - `继续搜标签`
  - `记住偏好`
- Reason text was expanded for common Discover tabs such as 同城 / 活动 / 话题 / 直播.
- The interaction remains a lightweight local discovery layer, not a recommendation platform.

### 3.2 Chat

Changed runtime files:

- `apps/flutter_elitesync_module/lib/features/chat/presentation/pages/chat_room_page.dart`
- `apps/flutter_elitesync_module/lib/features/chat/presentation/pages/conversation_list_page.dart`
- `apps/flutter_elitesync_module/lib/features/chat/presentation/widgets/icebreaker_card.dart`
- `apps/flutter_elitesync_module/test/features/chat/presentation/pages/chat_room_page_test.dart`
- `apps/flutter_elitesync_module/test/features/chat/presentation/pages/conversation_list_page_test.dart`

What changed:

- Conversation list now includes a `聊天节奏` guidance entry and a `回聊 / 稍后再聊` guidance card.
- The sheet and card both explain that this is local guidance, not a persistent queue.
- Chat room now includes:
  - `关系摘要`
  - `语音节奏`
  - `稍后再聊`
  - `AI 续话 / 低压建议`
  - `冷场恢复`
- Icebreaker wording was aligned so `稍后再聊` and `冷场恢复` are presented as candidate positions rather than durable system queues.
- The low-pressure draft helper only supplies local copy suggestions and does not auto-send anything.

### 3.3 Me / Appearance

Changed runtime files:

- `apps/flutter_elitesync_module/lib/features/profile/presentation/pages/profile_page.dart`
- `apps/flutter_elitesync_module/test/features/profile/presentation/pages/profile_page_test.dart`

What changed:

- Added `个人经营区` as a more readable operating hub.
- Added `标签表达体系` to group profile tags into slow-dating-friendly expression sections.
- Expanded `AI 助理 / 展示建议` into actionable display guidance.
- Added `AI 草稿助手` for local self-introduction and greeting drafts.
- Added `轻语音表达` as a voice-card candidate slot with a pre-recording explanation.
- Added `个人空间外观` as a preview-only presentation layer.
- The profile truth chain remains service-side; the new UI is display and expression only.

### 3.4 Settings

Changed runtime files:

- `apps/flutter_elitesync_module/lib/features/profile/presentation/pages/settings_page.dart`
- `apps/flutter_elitesync_module/test/features/profile/presentation/pages/settings_page_test.dart`

What changed:

- Added `个人空间外观` under the display / appearance grouping.
- The bottom sheet explains the boundary between preview-layer appearance and the real settings center.
- Permission explanation is kept before any future image/background style path.
- The admin verification subtitle wording was aligned from queue language to list/status language.

### 3.5 Supporting Copy Alignment

Additional wording alignment was applied to keep the 5.3 slice consistent across pages:

- `稍后再聊` is used consistently instead of older queue-oriented wording.
- `冷场恢复` is treated as a low-pressure candidate path.
- `个人空间外观` is treated as a preview layer.
- `设置中心` remains the real account / privacy / safety center.
