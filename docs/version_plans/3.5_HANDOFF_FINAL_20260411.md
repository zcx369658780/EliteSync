# 3.5 最终交接稿

更新时间：2026-04-11  
版本：`3.5 / Astro Engine MVP / 参数化基础版`

## 当前结论

`3.5` 已完成并可正式归档。  
本版目标不是完整专业占星工作台，而是先做出一个**可控、可回归、可扩展**的 Astro Engine MVP。

## 这版做了什么

### 1. Engine / Schema

- 星盘响应补充了 `engine_info`
- 星盘响应补充了 `metadata`
- canonical / derived / display-only 的边界更清楚
- `chart_data` 保持兼容，不破坏旧读取逻辑

### 2. 参数工作台

- 盘面设置页收成了工作台入口
- 参数面板提供最小可用的切换：
  - 黄道制
  - 宫位制
  - 相位密度
  - 容许度
  - 点位范围
- 预设、恢复默认和参数说明可直接使用

### 3. 盘面精细绘制

- 本地 SVG 盘面完成标签避让
- 相位线完成层次与密度控制
- 盘心与环带阅读性更稳定
- 星盘页保留只读摘要，避免把页面做成二次参数工作台

### 4. 参数解读与样例对照

- 增加了参数解读最小版
- 增加了固定样例对照
- 生成了差异报告，可重复回归

### 5. 阶段 5 收口

- smoke / regression / rollback 检查完成
- 关键页面截图已整理
- 验收报告已完成
