# EliteSync 2.9 Operations SOP

更新时间：2026-04-08

## 1. 日常职责

- 查看 health
- 查看版本检查
- 查看举报处理队列
- 查看认证审核队列
- 查看用户封禁 / 申诉状态

## 2. 处理优先级

### P0

- 登录异常
- 首页白屏
- 资料保存失败
- 消息页不可达
- 匹配页不可达

### P1

- 举报堆积
- 审核队列堆积
- 用户封禁 / 申诉等待过长

### P2

- 文案微调
- 后台列表筛选优化

## 3. 标准动作

- 封禁前确认举报证据
- 驳回前写清理由
- 申诉处理要保留备注
- 处理完成后复查前台状态

## 4. 升级路径

- 运营处理不了 -> 管理员
- 管理员处理不了 -> 回滚 / Hotfix
