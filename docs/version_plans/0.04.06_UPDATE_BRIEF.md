## 主要更新

1. 完成 4.6P 音频播放链定向修正版收口：LiveKit 真语音已进入可听闭环，手机端发言可被电脑端模拟器实际听到，音轨可视化、播放链与音频路由已纳入主链。
2. 继续保持 4.6 RTC 1v1 语音链路、消息主链、通知中心、图片消息与视频消息等既有能力，不扩成多人通话、直播或视频动态平台。
3. 版本号、版本检查、关于页历史、发布配置与 APK 文件名统一到 `0.04.06 / 40600`，并将 4.6P 的更新内容纳入 app 内发版说明。

## 发版口径

- 继续保持 4.0 媒体 / 存储 / 附件管线、4.1 非官方人格问卷、4.2 图片消息主链、4.3 动态流、4.4 视频消息、4.4S 媒体链稳定性与 4.5 通知增强边界不变。
