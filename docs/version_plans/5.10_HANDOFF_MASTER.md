## 5. 解释层治理与用户控制说明

5.10 的治理是 Settings presentation 内的用户说明与本地控制入口。

三类能力说明：

- `关系解释提示`：帮助理解 Match 中为什么值得聊，不改变匹配算法。
- `个人表达建议`：帮助整理个人页表达，不会自动修改资料。
- `聊天开场建议`：帮助准备可编辑草稿，不会自动发送消息。

当前控制策略：

- 采用说明型开关占位。
- 文案为 `本机控制 · 后续完善`。
- 不持久化到服务端。
- 不写本地偏好。
- 不影响实际 Match / Profile / Chat 页面显示逻辑。
