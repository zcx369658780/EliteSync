# 修改后回归验收清单

> 用于所有修改后的最小回归。涉及高风险模块时必须完整执行，并在验收报告中记录结果。

## 1. 基础检查

- [ ] 应用能启动
- [ ] 登录能成功
- [ ] 版本号显示正确
- [ ] 主要页面可进入
- [ ] 无致命空白页 / 崩溃

## 2. 数据库相关

- [ ] 新增/修改数据能落库
- [ ] 旧数据未被意外覆盖
- [ ] 初始化逻辑可重复执行
- [ ] 迁移脚本可回滚
- [ ] 备份可生成
- [ ] 恢复可成功

## 3. 地图 / 定位 / 经纬度

- [ ] 定位权限申请正常
- [ ] 地点搜索有结果
- [ ] 结果包含可识别的名称 / 地址
- [ ] 需要时可保留经纬度
- [ ] 保存后经纬度仍可回读
- [ ] 逆地理编码或地理编码不再静默失败

## 4. 路由 / 页面跳转

- [ ] 入口仍可到达
- [ ] 返回栈正常
- [ ] 页面切换无黑屏
- [ ] 生命周期触发正常
- [ ] 新旧入口映射正确

## 5. 状态管理 / 缓存

- [ ] 保存后状态回写正确
- [ ] 刷新后状态不丢失
- [ ] 退出重进后状态可恢复
- [ ] 旧缓存不会覆盖新值

## 6. 按钮 / 事件

- [ ] 按钮触发正确
- [ ] 点击后有反馈
- [ ] 选中态正确
- [ ] 禁用态正确
- [ ] 旧按钮行为未丢失

## 7. 配置 / SDK / 权限

- [ ] 环境变量加载正确
- [ ] 第三方 SDK 初始化正常
- [ ] 权限链路正常
- [ ] 运行时配置与仓库文档一致

## 8. 版本 / 发布

- [ ] changelog 已更新
- [ ] 版本检查接口正确
- [ ] 下载地址可访问
- [ ] 发布脚本可跑
- [ ] 回滚脚本可跑

## 9. 高风险模块专检

- [ ] 数据库 / 迁移
- [ ] 地图 / 定位
- [ ] 路由 / 生命周期
- [ ] 状态管理
- [ ] 配置 / SDK
- [ ] 备份 / 恢复

## 10. 交付判定

- [ ] 已验证所有高风险变更
- [ ] 已确认未验证项
- [ ] 已确认回滚点
- [ ] 可允许交付 / 不允许交付
